# Автогенератор дашбордов GetCourse

Настольное приложение, которое автоматически создаёт аналитический дашборд в GetCourse по параметрам вебинара — без ручного кликанья по интерфейсу.

---

## Проблема

После каждого вебинара нужно вручную заходить в GetCourse, создавать дашборд и добавлять виджеты с нужными фильтрами — группами участников, предложениями, UTM-метками. При большом количестве вебинаров это рутинная, повторяющаяся работа, которая отнимает время и в которой легко ошибиться: опечататься в названии группы, перепутать фильтр, забыть виджет.

---

## Для кого

Технические специалисты и маркетологи онлайн-школ, которые регулярно проводят вебинары на платформе GetCourse и ведут аналитику по каждому из них.

---

## Основная функция

**На входе** (вводится один раз в форму приложения):
- URL аккаунта GetCourse
- E-mail и пароль администратора
- Тег вебинара (например: `zero-luxury-8`)
- Дата вебинара
- Название предложения из GetCourse
- UTM-метка канала спикера (`utm_medium`)
- Имя Telegram-бота
![окно ввода данных](images/программа.jpeg)
**Что происходит внутри:**
1. Приложение открывает браузер и логинится в GetCourse
2. Создаёт новый дашборд с именем `Вебинар [тег]`
3. Последовательно добавляет виджеты с нужными фильтрами
4. Сохраняет дашборд и оставляет браузер открытым для проверки
![cjplfyysq lfi,jhl](images/результат.jpeg)
**Инструменты:** браузерная автоматизация через Playwright (без API GetCourse, работает через интерфейс).

---

## Формат результата

Готовый дашборд в GetCourse с тремя виджетами:

| Виджет | Источник | Что считает |
|--------|----------|-------------|
| Регистрации всего | Пользователи | участники группы `[тег] Регистрация на вебинар` |
| Посещения всего | Пользователи | участники группы `[тег] Посещение` |
| Платное участие | Заказы | заказы по указанному предложению |

Браузер остаётся открытым — можно сразу проверить результат.

## Функционал

### Реализовано (версия 1)
- Графический интерфейс (GUI) — не нужна командная строка
- Автологин в GetCourse
- Создание дашборда и трёх основных виджетов
- Сохранение настроек аккаунта между запусками
- Режим **«Разведка формы»** — диагностика для отладки: сохраняет скриншоты и список элементов интерфейса в папку `debug/`
- Лог выполнения в окне приложения

### Планируется
- Расширенный набор виджетов (UTM-источники, Telegram-бот, канал спикера, кнопки mini-app)
- Пакетное создание дашбордов для нескольких вебинаров сразу
- Экспорт итоговых цифр в таблицу

## Устранение ошибок

**Ошибка логина** — проверьте URL (без `/` в конце), e-mail и пароль.  
**Виджет создался без фильтра** — название группы в GetCourse должно точно совпадать с шаблоном `[тег] Регистрация на вебинар`.  
**Браузер завис** — закройте его вручную, нажмите кнопку снова.  
**Что-то пошло не так** — в папке `screenshots/` лежат скриншоты каждого шага, по ним видно где остановилось.
