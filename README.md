# GetCourse — Автосоздание дашборда

Настольное приложение для автоматического создания аналитического дашборда в GetCourse.  
Заполняешь параметры вебинара один раз — скрипт сам логинится, создаёт дашборд и добавляет виджеты.

---

## Что создаётся

Три виджета в дашборде:

| # | Название | Источник | Фильтр |
|---|----------|----------|--------|
| 1 | Регистрации всего | Пользователи | В группе `[тег] Регистрация на вебинар` |
| 2 | Посещения всего | Пользователи | В группе `[тег] Посещение` |
| 3 | Платное участие | Заказы | Предложение (по названию) |

---

## Установка (один раз)

**Требования:** Python 3.10+

```bash
pip install -r requirements.txt
playwright install chromium
```

---

## Запуск

```bash
python main.py
```

Откроется окно приложения.

---

## Заполнение полей

### Блок «Аккаунт GetCourse»

| Поле | Что вводить | Пример |
|------|-------------|--------|
| URL аккаунта | Адрес школы без слеша в конце | `https://myschool.getcourse.ru` |
| E-mail | Логин администратора | `admin@myschool.ru` |
| Пароль | Пароль администратора | `••••••••` |

### Блок «Параметры вебинара»

| Поле | Что вводить | Пример |
|------|-------------|--------|
| Тег вебинара | Уникальный тег, используется в названиях групп | `zero-luxury-8` |
| Дата вебинара | Дата в формате ДД.ММ.ГГГГ | `15.06.2026` |
| Предложение | Полное название предложения из GetCourse | `[zero-luxury-8] Платное участие — 499 руб.` |
| Спикер (utm_medium) | Значение utm_medium для канала спикера | `channel_speaker` |
| Telegram-бот | Юзернейм бота (без @) | `zerocoder_university_bot` |

> Нажми **«Сохранить настройки»** — URL, e-mail, пароль и бот сохранятся на следующий запуск.  
> Тег, дату и предложение нужно вводить каждый раз (они меняются под каждый вебинар).

---

## Порядок работы

1. Запусти `python main.py`
2. Заполни все поля
3. Нажми **«Создать дашборд»**
4. Откроется браузер — скрипт работает автоматически, не трогай его
5. Дождись сообщения в логе: `✓ Готово! Дашборд создан`
6. Проверь результат в браузере (он остаётся открытым)
7. Закрой браузер вручную

---

## Кнопка «Разведка формы»

Используется для диагностики, если что-то сломалось в GetCourse.  
Создаёт тестовый дашборд **`ДИАГНОСТИКА_УДАЛИТЬ`** и сохраняет в папку `debug/` скриншоты и списки элементов формы.

> После диагностики удали дашборд `ДИАГНОСТИКА_УДАЛИТЬ` вручную в GetCourse.

## Если что-то пошло не так

**Скрипт упал с ошибкой** — посмотри лог в нижней части окна, там будет текст ошибки.  
Дополнительно в папке `screenshots/` лежат скриншоты каждого шага — по ним видно где именно остановилось.

**Виджет создался без фильтра** — проверь, что название группы в GetCourse точно совпадает с шаблоном `[тег] Регистрация на вебинар` (с квадратными скобками и пробелами).

**Ошибка логина** — проверь URL (без слеша в конце), e-mail и пароль.

**Браузер открылся но завис** — закрой его вручную, нажми кнопку снова.
